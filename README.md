## 🎯 Objetivo do Projeto
Este projeto tem como objetivo aplicar o pipeline de Machine Learning, desenvolvendo:

- Um modelo de **Classificação**
- Um modelo de **Regressão**

---

## 📁 Dataset Utilizado
Foi utilizado o dataset **"Predict Customer Purchase Behavior"** disponível no Kaggle.

### 🔎 Problema abordado:
Prever se um cliente irá realizar uma compra com base em características como:

- Idade  
- Gênero  
- Renda anual  
- Número de compras  
- Tempo no site  
- Programa de fidelidade  
- Descontos utilizados  

👉 Variável alvo (target):  
**PurchaseStatus**
- 0 = Não comprou  
- 1 = Comprou  

---

## ⚙️ Pipeline de Machine Learning

### 1. Coleta e Seleção dos Dados
- Dataset obtido do Kaggle

### 2. Preparação dos Dados
- Verificação de valores nulos  
- Remoção de registros duplicados  
- Análise dos dados  

### 3. Divisão dos Dados
- Separação em conjunto de treino e teste  

### 4. Treinamento dos Modelos

#### 📌 Classificação
- Regressão Logística  
- KNN (K-Nearest Neighbors)  

#### 📌 Regressão
- Regressão Linear  

---

## 📈 Avaliação dos Modelos

### 🔹 Classificação
Métricas utilizadas:
- Accuracy  
- Precision  
- Recall  
- F1-Score  
- Matriz de Confusão  

---

### 🔹 Regressão
Métricas utilizadas:
- MAE  
- MSE  
- RMSE  
- R²
